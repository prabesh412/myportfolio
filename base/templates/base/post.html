Nepse Data Scraper

This was a project I did in november when I got a request from fellow reddit friend to help him with data scraping . I was the sole creator/developer of this project.

The system stored scraped data from mero lagani website which will be crawled by a powerful python spider created using scrapy.

Fields like date and time, Share unit, Price , LTP will be provided for the user by this Scraper.

Data can be further uploaded to excel format or json format.

I worked on various project like this using scraper and web drivers.

Technologies:
  • - Python
  • - Scrapy
  • - Selenium
  • - Postgres
